Selektor element

Selektor element odnosi się do wszystkich elementów HTML określonego typu

Przykładowo, selektor code odnosi się do wszystkich elementów <code> na stronie

Jest to szczególnie przydatne, gdy mamy element taki jak <code>, któremu chcemy wszędzie nadać ten sam styl

Selektor element1, element2

Selektor element1, element2 działa podobnie jak wcześniej wspomniany selektor element. Zamiast wybierać każdy element do zmiany osobno, można użyć tego selektora.

Chcąc zaznaczyć jednocześnie <h1> oraz <h2> możemy użyć selektora h1, h2 zamiast używania dwóch selektorów element

Selektor element1 element2

Selektor element1 element2 odnosi się do elementów HTML element2, które są zagnieżdżone w elemencie element1

Taki selektor przydaje się, gdy chcemy zastosować styl do elementów element2 znajdujących się tylko w elementach element1 i nigdzie indziej

Selektor element1 > element2

Selektor element1 > element2 działa bardzo podobnie jak poprzedni, z tym że element2 musi być bezpośrednio dzieckiem elementu element1

Warto dodać, że takiego samego selektora możemy także używać z klasami, bądź ID. Dzięki temu można tworzyć konstrukcje takie jak .class1 > element1
