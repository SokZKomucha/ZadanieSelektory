Co to ID?

ID to atrybut w HTML i CSS, który identyfikuje unikalny element na stronie internetowej

Od klas się różni tym, że każdy element na stronie może mieć tylko jedno ID i jest to ważne ze względu na niepowtarzalność i jednoznaczność identyfikacji elementu

Selektor #id

W CSS selektor ID jest używany za pomocą #id

Taki selektor można wykorzystać do zastosowania stylu tylko do określonego elementu posiadającego takie samo ID.

Potencjalne zastosowania ID

ID w samym HTMLu, poza CSSem, ma wiele zastosowań

Bardzo przydatne jest, gdy pracujemy w JavaScripcie z różnymi interaktywnymi elementami na stronie. Wtedy każdy element na stronie może być oddzielnie kontrolowany, tym samym nie ingerując z innymi
